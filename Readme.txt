How to Use

1. Open Main.go in your chosen IDE
2. Grab your CapMonster API Key from https://capmonster.cloud/Dashboard and Place your API Key in the apiKey Variables
3. Grab your ReCaptcha V2 SiteKey and Place it in the siteKey Variable
4. Grab the site URL which you wish to generate a ReCaptcha Response for and place it in the reqURL variable
